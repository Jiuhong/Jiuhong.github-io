原理解释（简单）

网络结构

基础网络：

  • 该网络结构借鉴了GoogLeNet，24个卷积层，2个全连接层。

不同之处：

  • 使用1x1 reduction layers紧跟 3×3 convolutional layers 取代Goolenet的 inception modules

检测原理

输出的数据意义

  • 前两个数值表示最终将图片划分的单元数7x7
  • 最后一个数值30是由20+5x2来计算出的，20代表分类数目，5分别代表（置信度， x， y， w， h），2代表每个单元检测的box数目。

损失值

损失值需要计算四类：

  • 坐标预测损失
  • 置信度预测损失（含bojiect的box）
  • 置信度预测损失（不含bojiect的box）

网络结构构架

与论文中不同之处：

  • 添加了pad函数使初始图像扩充到454x454，可能是为了完整保留图像信息
  • 多添加了一个全链接层，可能是为了减少参数
  • 各网络层参数发生了变化

损失值计算

损失值主要根据IOU并交比和标签的单元值respose来计算含目标和不含目标的损失值

思考

对于坐标损失只计算含目标的单元区

对于准确率confidence需要计算含目标和不含目标的两类损失值

  • 预测出不存在的目标时，该单元损失值在不含目标中计算
  • 未预测到正确目标时， 该损失值在不含目标中计算

对于分类损失只计算含目标的单元数

数据处理

数据处理要点:

  • 每个图像标签用一个字典来存储，包含图像路径，翻转和边框信息
  • 使用列表来存储字典，使用pickle包来保存列表文件

思考：

  • 在数据处理中可以看出每个单元区域只能标注一个目标，其它的目标则会丢失，因此yolo_v1对目标密集区域的检测效果不太好。
